# Santa Survival Game

A side-scrolling web game where you control Santa to deliver presents to houses while dodging missiles shot by Washington.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open your browser and navigate to `http://localhost:8080`

## How to Play

- Move your mouse cursor up and down to control Santa's vertical position
- Santa stays locked to the left side of the screen
- Fly over houses to deliver presents and score points
- Avoid missiles shot by Washington - you have 3 lives
- Game over when you run out of lives

## Controls

- Mouse movement: Control Santa's Y position
